# Expense Tracker using Python

This project is a simple command-line based Expense Tracker built using Python. It helps users record, manage, and view their daily expenses.

## Features
- Add new expenses with amount and category
- View all recorded expenses
- Calculate total spending
- Simple and user-friendly command-line interface

## Technologies Used
- Python
- Basic data structures (lists and dictionaries)

## How it Works
- The user selects an option from the menu:
  1. Add Expense
  2. View Expenses
  3. Exit
- When adding an expense, the user enters:
  - Amount
  - Category (e.g., Food, Travel, Shopping)
- The program stores the data and displays all expenses along with the total amount spent
